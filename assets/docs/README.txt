=================================================
          AirBar Installer - README
=================================================

Bienvenue dans AirBar Installer !

Cette application facilite l'installation et la gestion
du serveur AirBar pour les aéro-clubs.

=================================================
DÉMARRAGE RAPIDE
=================================================

1. Lancez l'application AirBar Installer
2. Suivez l'assistant d'installation pas à pas
3. Configurez votre serveur
4. Accédez au dashboard de gestion

=================================================
FONCTIONNALITÉS
=================================================

✓ Installation automatique de Docker
✓ Installation automatique de Git
✓ Configuration simplifiée du serveur
✓ Dashboard de gestion intuitif
✓ Visualisation des logs en temps réel
✓ Système de sauvegardes
✓ Mises à jour automatiques
✓ Désinstallation propre
