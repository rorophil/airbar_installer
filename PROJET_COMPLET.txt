⚠️ AVANT DE COMPILER
=================================================

1. Créer l'Icône de l'Application
   ----------------------------------------
   L'icône app_icon.ico est REQUISE pour:
   • L'exécutable Windows
   • L'installateur Inno Setup
   • Les raccourcis

   OPTIONS:
   
   Option A - Icône temporaire (rapide):
   □ Copiez: windows\runner\resources\app_icon.ico
   □ Vers: assets\images\app_icon.ico
   
   Option B - Icône personnalisée:
   □ Suivez: assets\images\ICON_GUIDE.txt
   □ Créez une icône multi-résolution (16x16 à 256x256)
   □ Format .ico requis
   □ Thème suggéré: Avion + Verre (bleu/orange)
   
2. Vérifier les Prérequis
   ----------------------------------------
   □ Flutter SDK installé et configuré
   □ Visual Studio 2022 ou Build Tools
   □ Windows 10/11 64-bit
   □ Inno Setup 6 (pour créer l'installateur)

=================================================
🚀 PROCÉDURE DE BUILD
=================================================

Étape 1: Créer l'Icône
----------------------------------------
Voir section "AVANT DE COMPILER" ci-dessus

Étape 2: Build Automatique
----------------------------------------
Ouvrez PowerShell et exécutez:

  .\build.ps1

Ce script va automatiquement:
✓ Nettoyer les builds précédents
✓ Récupérer les dépendances
✓ Analyser le code
✓ Compiler l'application Windows
✓ Vérifier les fichiers de sortie
✓ Compiler l'installateur Inno Setup (si présent)

Étape 3: Résultat
----------------------------------------
Si tout réussit, vous aurez:

• Exécutable:
  build\windows\x64\runner\Release\airbar_installer.exe
  
• Installateur (si Inno Setup installé):
  build\windows\installer\AirBar_Installer_Setup_1.0.0.exe

=================================================
📋 OPTIONS DU SCRIPT DE BUILD
=================================================

Build complet (défaut):
  .\build.ps1

Build sans nettoyage préalable:
  .\build.ps1 -Clean:$false

Build sans tests:
  .\build.ps1 -SkipTests

Build application uniquement (sans installateur):
  .\build.ps1 -BuildInstaller:$false

=================================================
🔧 DÉPANNAGE
=================================================

Problème: flutter doctor montre des erreurs
Solution: Installez Visual Studio Build Tools

Problème: build.ps1 ne peut pas s'exécuter
Solution: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned

Problème: Inno Setup non trouvé
Solution: Installez depuis https://jrsoftware.org/isdl.php

Problème: Icône manquante
Solution: Copiez l'icône par défaut ou créez-en une
